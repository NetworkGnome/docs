CHIP (Chiodo Infrastructure Platform) is an enterprise infrastructure operations platform that unifies documentation, management, and operational procedures for modern hybrid infrastructure. Whether you're navigating network topology, managing Proxmox virtualization clusters, operating Docker-based services, or executing disaster recovery procedures, CHIP serves as the authoritative reference for every layer of the stack.

## What CHIP Covers

CHIP is organized into two major areas:

**Platform** — the reference documentation for infrastructure design and configuration:
- **Network**: VLAN design, DNS zones, firewall policy, and routing topology
- **Compute**: Proxmox hypervisor configuration, VM and LXC management, and storage architecture
- **Services**: Docker runtime environments, service stack catalog, and ingress/reverse proxy configuration
- **Observability**: Monitoring infrastructure, alerting rules, and log aggregation pipelines
- **Security**: Access control policies, certificate management, and security hardening standards

**Operations** — the procedural knowledge for day-to-day and incident operations:
- **Runbooks**: Step-by-step guides for common tasks and maintenance windows
- **Disaster Recovery**: Backup strategies, retention policies, and restoration procedures
- **Automation**: Ansible playbook reference and scripting standards
- **Standards**: Naming conventions, change management process, and engineering guidelines
